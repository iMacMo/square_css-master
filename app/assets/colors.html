Short Info about color classes.

Classes
sq-text-base - This variable is using the base text color.
sq-text-white - This variable is using the white text color. Added black background color since text color is white.
sq-primary - This variable is using white text and the primary background color.
sq-primary-highlight - This is using the primary highlight variable, ment for highlighting objects.
sq-secondary - This variable is using white text and the secondary background color.
sq-secondary-highlight - This is using the secondary highlight variable, ment for highlighting objects.
sq-success - This variable is using white text and the success background color.
sq-success-highlight - This is using the success highlight variable, ment for highlighting objects.
sq-info - this variable is using white text and the info background color.
sq-info-highlight - This is using the info highlight variable, ment for highlighting objects.
sq-warning - This variable is using white text and the warning background color.
sq-warning-highlight - This is using the warning highlight variable, ment for highlighting objects.
sq-danger - This variable is using white text and the danger background color.
sq-danger-highlight - This is using the danger highlight variable, ment for highlighting objects.
sq-background - This variable is using the background color.
sq-background-secondary - This variable is using the secondary background color.
sq-black - This variable is using the black text color.
sq-grey - This variable is using the grey text color.
sq-white - This variable is using the white background color.
sq-muted - This variable is using the muted text color.
sq-muted-highlight - This is using the muted highlight variable, ment for highlighting muted objects.
sq-divider - This is using the divider variable, ment for changing color on for example an hr tag.
