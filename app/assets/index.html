Utility

Utility classes can be used to created margin, padding and borders
Important!: Border-style classes can only be added on sq-border-1, 3 or 5.

Classes
sq-air-y-05 - Creates margin on the top and bottom. (0.5em, 1em,2em,3em,4em & 5em)
sq-air-x-05 - Creates margin on the left and right.
sq-margin-05- Creates margin on all sides.
sq-mt-05 / sq-mb-05- Creates margin only on top (-mt) or bottom (-mb).
sq-padding-05 - Creates padding on all sides. (0.5em, 1em,2em,3em,4em & 5em)
sq-l-padding-05 / sq-r-padding-05 - Creates padding only on the left / right. (Solid only)
sq-t-padding-05 / sq-b-padding-05 - Creates padding only on top / bottom. (Solid only)
sq-border-1 - Creates a border on all four sides. (1px, 3px, 5px)
sq-t-border / sq-b-border - Creates a border only on top / bottom. (Solid only)
sq-l-border / sq-r-border - Creates a border only on the left / right. (solid only)
sq-dashed (Border styles) - Changes the borderstyle (solid is default), (sq-dashed, dotted, groove, none, inherit or ridge)

Typography

Examples and info about Typography classes

Font classes
sq-bold - Makes your text bold.
sq-sans-serif - Changes your font to sans-serif.
sq-cursive - Makes your text cursive.
sq-fantasy - Makes your text look thicker and bolder.

Text Size classes
sq-text-small - Makes your text look smaller.
sq-text-large - Makes your text look bigger.

Colored Text
sq-text-muted - Changes your text color to "Muted"
sq-text-success - Changes your text color to "Success"
sq-text-warning - Changes your text color to "Warning!"
sq-text-danger - Changes your text color to "Danger!"

Text Tranform-modifiers
sq-text-uppercase - All letters will appear in uppercase.
sq-text-lowercase - All letters will appear in lowercase.
sq-text-capitalize - First letter will appear as a uppercase letter.
Dropcaps - Makes first letter bigger than the rest of the text, see here for demo. Note That you might have to do some adjustments to the CSS.

Text-Decoration Lines
sq-overline - Creates a overline above your text.
sq-linethrough - Creates a line through your text.
sq-underline Creates a line under your text.

Forms / input fields

forms are are for allowing the user to input various forms of information to be passed along by scripts etc. forms are scoped under the class sq-form, and styling can be changed in the form.css.

CSS elements
.sq-form - used to limit the scope of the form styling.
Fieldset - tag used for grouping together elements within the form.
Legend - tag works as a headline for the elements within the fieldset.
Form - Root element for forms,insert .sq-form to add form scope.
Label - Label tag used to label each input field.
Input - Input fields used to gather user information.

Example: name and date form elements

Personalia 
		
<form class="sq-form" action="index.html" method="post" type=text>
	<fieldset>
		<legend>Personalia</legend>
		<label>First Name</label>
		<input type="text" name="" value="" placeholder="First name">
		<label>Last name</label>
		<input type="text" name="" value="" placeholder="Last name">
		<label>Date of birth</label>
		<input type="date" name="" value="">
	</fieldset>
</form>
